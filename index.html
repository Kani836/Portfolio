Projects

Obstacle-Avoiding Car

This project involves designing a robotic car that can detect and avoid obstacles using ultrasonic sensors. The car continuously moves forward and, upon detecting an obstacle, intelligently determines the safest direction to turn, ensuring smooth navigation. The integration of servo motors allows the ultrasonic sensor to scan the environment effectively, enhancing the car's decision-making capabilities. This project demonstrates my skills in embedded systems, Arduino programming, and real-time sensor data processing.

Automatic Water Level Controller

The automatic water level controller is designed to efficiently manage water levels in storage tanks. Using ultrasonic sensors, the system continuously monitors the water level and automatically controls the motor to prevent both overflow and dry running. This project highlights my ability to implement automation techniques, work with microcontrollers, and develop energy-efficient systems with minimal human intervention.
